# Sleep Pattern Learner

## Setup

```bash
npm install
```

## Development

```bash
npm start
npm run android
npm run ios
```

## Testing

```bash
npm test
npm run typecheck
npm run lint
```

## Features I've Covered

- Sleep logging with timer and manual entry
- Timeline and trend charts
- EWMA-based wake window learning
- Dynamic schedule generation
- Local notifications
- Contextual coaching tips
